# RedotRepeater

## Purpose

Provides the behavior behind `<x-repeater>`.

## What It Handles

- inserting items
- removing items
- clearing the list
- synchronizing the hidden JSON value
- optional sortable behavior
- optional confirmation behavior

## Used By

- `repeater` init
- `<x-repeater>`
