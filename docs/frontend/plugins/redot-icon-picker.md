# RedotIconPicker

## Purpose

Provides the browser-side behavior behind `<x-icon-picker>`.

## What It Handles

- opening the icon picker UI
- querying Font Awesome
- updating the selected icon value
- updating the preview icon

## Used By

- `icon-picker` init
- `<x-icon-picker>`
