# Coloris Init

Initializer name:

- `coloris`

Option prefix:

- `coloris-*`

Default options:

- `rtl` from page direction
- `themeMode` from current theme key
- `theme = 'polaroid'`
- `formatToggle = true`

Special behavior:

- assigns a generated `coloris-id` when missing
- stores the instance in jQuery data under `coloris`
