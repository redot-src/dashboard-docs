# Tempus Dominus Init

Initializer name:

- `tempus-dominus`

Option prefix:

- `date-*`

Default options:

- theme follows current dashboard theme
- calendar enabled
- clock disabled
- default format `yyyy-MM-dd`
- default hour cycle `h12`

Special attributes:

- `datetime`
- `only-time`

Stores instance under:

- `picker`
