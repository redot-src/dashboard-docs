# TinyMCE Init

Initializer name:

- `tinymce`

Default behavior:

- language from `<html lang>`
- directionality from `<html dir>`
- height `300`
- no menubar
- no branding
- dark/light skin based on current theme
- image uploads to `/tinymce/upload`

Built-in editor features:

- lists
- links
- tables
- image insertion
- alignment
- text direction controls

Stores instance under:

- `tinymce`
