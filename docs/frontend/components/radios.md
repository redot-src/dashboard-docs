# Radios Component

Component:

```blade
<x-radios />
```

## Common Inputs

- `id`
- `name`
- `title`
- `value`
- `options`
- `disabled`
- `inline`
- `validation`
- `required`
- `hint`

## Behavior

- checks the current selected value
- supports inline or block display
- applies grouped validation metadata to the first radio
