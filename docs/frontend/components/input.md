# Input Component

Component:

```blade
<x-input />
```

## Purpose

Render a standardized text-like input field.

## Common Inputs

- `id`
- `name`
- `type`
- `title`
- `value`
- `required`
- `hint`
- `floating`
- `prepend`
- `append`

## Built-In Behavior

- floating label mode
- input-group prepend and append
- password visibility toggle when used for password fields
- standard `form-control` styling
