# Empty Component

Component:

```blade
<x-empty />
```

## Purpose

Render a friendly empty-state card.

## Props

- `icon`
- `title`
- `subtitle`

## Defaults

- icon: `fas fa-circle-xmark`
- title: `Nothing to show here`
- subtitle: localized fallback text
