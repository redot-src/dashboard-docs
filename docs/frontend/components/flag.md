# Flag Component

Component:

```blade
<x-flag />
```

## Props

- `code`
- `size`

## Defaults

- `code = 'eg'`
- `size = 'md'`

## Behavior

- loads Tabler flag CSS once
- renders a `flag-country-{code}` class
