# Rating Component

Component:

```blade
<x-rating />
```

## Common Inputs

- `id`
- `name`
- `title`
- `value`
- `stars`
- `size`
- `icon`
- `required`
- `hint`

## Behavior

- renders descending radio options
- uses CSS variable `--star-size`
- renders the configured icon for each star
